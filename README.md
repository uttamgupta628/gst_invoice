# GST Invoice Calculator

A free, client-side GST invoice calculator built with React + Vite.

## Features
- Add unlimited line items with HSN/SAC codes
- Choose GST rates: 0%, 5%, 12%, 18%, 28%
- Auto CGST/SGST split or IGST toggle
- GST slab-wise breakdown table
- Print to PDF via browser print

## Deploy to Vercel (free)

1. Push this folder to a GitHub repo
2. Go to vercel.com → New Project → Import your repo
3. Framework: Vite (auto-detected)
4. Click Deploy — done!

## Run locally

```bash
npm install
npm run dev
```
